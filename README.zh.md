# Synnia

> **From Chaos to Canon. Locally.**

[English](./README.md)

Synnia 是一款开源的**数字资产一致性引擎**与管理工具。在这里，我们将图片、文本或两者结合的产物统称为“数字资产”。

## 核心哲学

Synnia 的操作哲学深受节点化工作流启发，但有着本质的不同：

*   **Synnia (Asset-as-Node)**: 这里的节点代表**数字资产**本身（实体）。
*   **ComfyUI (Function-as-Node)**: 传统的节点代表**功能**或处理步骤。

## 为什么选择 Synnia？

Synnia 内置了多种 Agents，它们充当连接外部 AI 能力的桥梁。你可以通过标准接口调用任何你喜欢的模型进行文字或图片的生成与扩展。

Synnia 并不关注底层的生成实现细节，它的核心使命是提供一个**全局的（品牌级、IP级）视野**。作为一致性引擎，它确保在资产不断生成和拓展的过程中，始终维持风格、设定与逻辑的高度统一。

如果你是以下人群，Synnia 将是你管理和拓展数字创意宇宙的最佳伙伴：
*   独立开发者
*   品牌设计师
*   文学创作者
*   漫画家

## 核心特性

*   **一致性引擎 (Consistency Engine)**: 为所有扩展内容提供品牌/IP 级的一致性保障。
*   **模型无关 (Model Agnostic)**: 通过 Agents 接入任何你喜欢的 AI 模型接口。
*   **隐私优先 (Privacy First)**: 100% 本地存储，你的创意只属于你。
